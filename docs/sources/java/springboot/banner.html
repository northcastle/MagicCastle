banner.txt 文件 ​

SpringBoot 项目的启动日志中可以通过自定义 banner.txt 文件的内容修改启动日志中的标识。 本文就展示一个作者比较喜欢的图标 ： 【老鹰】。

文件位置 ​

banner.txt 文件在项目中的位置 ： 在项目的 resources 下。

java
ProjectName
    | -- src
        | -- main
            | -- java
            | -- resources
                | -- application.yaml
                | -- banner.txt # 这个就是我们要修改的文件
    | -- pom.xml
